Global File

Notes

1) Bot prefix is n! (e.g. n!debug)

2) The default Embed Color is Purple.


Startup Event

  • Prints ===> READY! <===
  • Parses Twitch accessToken on interval
  • Continuously polls Twitch API for Stream Status for various casters.

Message Event

Command Description Usage Permission
invite DMs the Bot's invite link. - @everyone
support DMs the Discord invite to the support server. - "
roadmap DMs the link to the Development Roadmap. - "
math Solves the given equation. No spaces allowed. n!math 2+2 "
serverinfo Provides convenient information about the server. - @guildOwner
spam Spams given message x times. n!spam <x> <message> @botOwner
setpresence Sets the activity of the client. n!setpresence <activity> "
eval Javascript Eval() command. n!eval <query> "
debug Shows client version and uptime. - "
